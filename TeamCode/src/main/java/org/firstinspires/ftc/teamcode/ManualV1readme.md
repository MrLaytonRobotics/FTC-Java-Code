Tests driving the Swyft robot.

L1/LB Stops all motors, goes into Dryrun mode (Motors are offline).
R1/RB Stops all motors, goes into Running mode (Motors are online).

A: Increases intake speed
B: Decreases intake speed (can go backwards)

X: Increases shooting speed (both motors)
Y: Decreases shooting speed (down to 0 minimum)

Dpad up: Increases drive train speed.
Dpad down: Decreases drive train speed.

Left stick -> Omni-moves the robot
Right stick -> X-axis only, rotates the robot, changing orientation.
